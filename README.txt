Neon Nights Concert - Event Landing Page

Overview: This is a static, single-page frontend for the Neon Nights Concert 2025, built with HTML5 and CSS3. It promotes a music festival with a neon aesthetic, featuring a navigation bar, event banner, details, schedule, performers, call-to-action, and footer. The design is mobile-responsive and beginner-friendly, using semantic HTML and Flexbox.

Folder Structure:





index.html: The main page with all sections.



style.css: Stylesheet with neon aesthetic and responsive design.



assets/images/: Placeholder for local images (currently using Unsplash URLs).

Features:





Semantic HTML5 structure with sections for navigation, banner, details, schedule, performers, call-to-action, and footer.



Neon aesthetic with glowing text, hover transitions, and gradient backgrounds.



Mobile-first responsive design with media queries (992px, 768px, 576px) for navbar, grids, and images.



Flexbox for layout (navbar, details, schedule).



CSS Grid for performers section (3-column on desktop, 1-column on mobile).



Unsplash images for banner and performers to ensure visual appeal.



Social media links (Facebook, TikTok, Instagram) in the footer.



No JavaScript, keeping the site static as per initial requirements.

Implementation Details:





Navigation: Fixed navbar with smooth hover effects and vertical stacking on mobile.



Banner: Full-screen hero with a darkened Unsplash background and glowing title animation.



Details: Flexbox grid for date, time, and location with hover scaling.



Schedule: Flexbox list with hover shadow effects.



Performers: 3-card grid with verified Unsplash images, fixed to display correctly.



Call-to-Action: Neon-styled button linking to registration.



Footer: Consistent across pages with contact info and social links.



Responsive Design: Media queries adjust font sizes, padding, and layouts for mobile and tablet screens.

How to Use:





Save index.html and style.css in the project folder.



Unsplash image URLs are used for the banner and performers; for offline use, download images and place them in assets/images/, updating URLs in index.html.



Open index.html in a browser to view.



Test responsiveness using browser developer tools or by resizing the window.

Notes:





Images are sourced from Unsplash for reliability; local images can be used by updating URLs.



The site is static, with no interactive features like forms or JavaScript.



Social media links are included as provided by the user.
